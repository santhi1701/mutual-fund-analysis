# MF Analytics Platform

## Project Overview

This project is part of the Bluestock Fintech Individual Capstone.

The objective is to build a complete Mutual Fund Analytics Platform using publicly available mutual fund data.

- Virtual environment configured.
- Required Python libraries installed.

- NAV history saved as CSV.
- Fund metadata extracted.
- Data quality checks completed.
- AMFI scheme codes validated.
